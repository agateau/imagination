Introduction to

Imagination is a lightweight DVD slide show creator developed in C with the GTK+2 toolkit and Cairo. Imagination is designed to be be fast and easy-to-use. Its user interface is clean and intuitive, and does not include any confusing or useless options by default. It requires the ffmpeg encoder to produce the movie file and libsox to handle the audio. Yes you don't need any other third-party software!
